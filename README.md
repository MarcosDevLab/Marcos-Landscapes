# Marcos Landscapes

Welcome to the Marcos Landscapes repository! This documentation provides a comprehensive overview of the landscape photography portfolio website.

## Overview
This portfolio website is designed to showcase the beauty of nature through stunning landscape photography. The site provides a user-friendly interface for visitors to explore various galleries, view images in high resolution, and learn about the photographer's journey.

## Features
- **Responsive Design**: The website is fully responsive and adapts to different screen sizes for optimal viewing on desktop and mobile devices.
- **Image Galleries**: Multiple galleries categorized by themes, such as mountains, forests, and oceans.
- **High-Resolution Images**: Visitors can view images in high resolution, ensuring that details are preserved.
- **About the Photographer**: A dedicated section to learn about Marcos, his passion for photography, and his creative process.
- **Contact Form**: An easy-to-use contact form for inquiries and bookings.
- **Blog Section**: Regular updates and articles related to landscape photography, tips, and personal experiences.

## Technologies Used
- **HTML5**: For semantic and structured content.
- **CSS3**: For styling the website and creating a visually appealing layout.
- **JavaScript**: For interactive elements and dynamic content.
- **Bootstrap**: A front-end framework for responsive design.
- **GitHub Pages**: To host the website directly from this repository.

## Installation
To run this project locally:
1. Clone the repository: `git clone https://github.com/MarcosDevLab/marcos-landscapes.git`
2. Navigate to the project folder: `cd marcos-landscapes`
3. Open the `index.html` file in your web browser.

## Usage
Simply navigate through the different galleries using the navigation bar at the top. Click on any image to view it in high resolution and read accompanying captions.
